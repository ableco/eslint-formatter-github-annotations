# eslint-formatter-github-annotations

An [ESLint](https://eslint.org/) formatter to report as GitHub Checks annotations

## Usage

``` shell
npm install eslint-formatter-github-annotations
eslint -f github-annotations example.js
```

## Testing

``` shell
npm run test
```
